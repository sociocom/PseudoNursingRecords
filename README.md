# NursingRecord_NERdataset
### PatientStatus.xml
SOAP式模擬看護記録のSセクションに患者状態アノテーションを実施した．  
アノテーションガイドラインは ps_AnnotationGuidline.md 参照.

### ChiefComplaint.xml
SOAP式模擬看護記録のSセクションのうち，身体的主訴を含む文のみに患者状態アノテーションを実施した．  
アノテーションガイドラインは cc_AnnotationGuidline.md 参照.
